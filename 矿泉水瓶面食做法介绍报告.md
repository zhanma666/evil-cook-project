# 矿泉水瓶面食做法介绍报告

## 1. 概述

根据产品需求，我们在邪修厨房项目中新增了"矿泉水瓶面食妙用"功能模块。该模块展示了如何利用常见的矿泉水瓶在面食制作中发挥重要作用，为用户提供创新、便捷且低成本的烹饪解决方案。

## 2. 功能特点

### 2.1 精准量具应用
将550ml矿泉水瓶作为厨房量具，替代传统秤具，实现精准配比：
- 4格的面粉就是100g（累计5瓶4格面粉就可以得到500g面粉）
- 1瓶盖酵母粉就是5g
- 6格的温水就是250g（从瓶底开始计数）

### 2.2 发酵辅助工具
利用矿泉水瓶创造理想的发酵环境，提升发酵效率：
- 恒温发酵法：将40度的温水装入500ml的矿泉水瓶，将其埋入面团中央，创造35度的恒温环境，发酵时间缩短至45分钟（常温情况下需要1.5个小时）
- 摇面发酵法：将面粉，酵母，温水装入瓶中摇30秒成团，倒出来后直接发酵，可以轻松省去揉面的步骤

### 2.3 摇面糊容器
使用矿泉水瓶作为容器，简化面食制作流程：
- 制作松饼配方：鸡蛋1个➕面粉100g➕牛奶150ml➕糖10g，将这些装入瓶中，摇晃1分钟后醒发20分钟，瓶口挤出来的面糊自然成圆形
- 南瓜馒头改良：将南瓜泥与面粉按照1:2的比例装瓶摇匀，加入半瓶盖酵母粉后二次摇晃
